# 合规指南

本文档列出 fund-scout 使用过程中必须遵守的合规底线、robots.txt 合规说明、免责声明和使用者责任。

## 合规底线

### 强制限速

| 规则 | 说明 |
|------|------|
| 最小请求间隔 | 0.5 秒，不可关闭 |
| 推荐请求间隔 | 1.0~2.0 秒（关键接口） |
| 随机抖动 | 每次请求随机 sleep 0.5~2.0 秒 |
| 连续失败熔断 | 连续 30 次失败自动停止 |
| 禁止并发 | 所有请求必须串行，禁止多线程/协程并发 |
| 单 IP 请求上限 | 天天基金 ~50 次/会话后可能被封 5~10 分钟 |

### 禁止绕过反爬

- 禁止使用代理池轮换 IP 规避限速
- 禁止伪造 Cookie / Session 绕过登录验证
- 禁止使用无头浏览器自动化（Selenium / Playwright）绕过 JS 渲染
- 禁止逆向加密接口参数
- 禁止绕过 robots.txt 中的 Disallow 规则

### 禁止商业转售

- 禁止将抓取的数据以任何形式进行商业转售
- 禁止构建与数据源平台（天天基金、证监会等）竞争的产品或服务
- 禁止将数据提供给第三方用于商业用途
- 数据仅限个人学习研究使用

## robots.txt 合规说明

### 天天基金 `fund.eastmoney.com`

天天基金网站未公开 robots.txt 限制爬虫的具体路径，但基于行业惯例和平台性质：

- 公开信息页面（基金主页、档案页、经理页）可访问
- API 接口（lsjz、rankhandler 等）属于平台内部接口，使用时需遵守限速规则
- 不得高频访问导致平台服务受影响

### 证监会 `eid.csrc.gov.cn`

证监会网站为政府公开信息披露平台：

- 季报披露信息属于法定公开信息，可依法查阅
- 搜索接口和 PDF 下载为公开服务，使用时需遵守限速规则
- 不得对政府网站造成过大访问压力

### 合规原则

1. **最小必要原则**：只抓取所需数据，不批量遍历全站
2. **限速原则**：遵守最小请求间隔，不并发、不绕过限速
3. **用途限制原则**：数据仅用于个人学习研究，不商业转售
4. **尊重平台原则**：如平台明确要求停止爬取，应立即停止

**数据来源声明**

**不构成投资建议**

本工具提供的数据和分析结果仅供个人学习研究参考，不构成任何形式的投资建议、投资咨询或投资推荐。基金投资有风险，过往业绩不代表未来表现。投资者应根据自身风险承受能力独立做出投资决策。

**数据准确性**

本工具通过自动化方式从公开渠道获取数据，可能存在以下问题：
- 数据延迟：公开渠道数据更新可能滞后于实际交易
- 解析错误：HTML/PDF 解析可能因页面结构变化而出错
- 字段缺失：部分基金可能缺少某些字段数据
- 数据不一致：不同数据源之间可能存在数值差异

用户应以官方源数据为准，本工具数据仅供参考。

**禁止商业使用**

本工具及产出数据禁止用于以下用途：
- 商业数据转售

## 使用者责任

### 必须遵守

1. **遵守限速规则**：不降低 rate_limit 至 0.5s 以下，不关闭限速
2. **遵守数据用途限制**：仅用于个人学习研究，不商业转售
3. **遵守法律法规**：遵守所在地区关于数据获取和使用的法律法规
4. **人工抽检**：对抓取结果随机抽检 2~3 只与源网页对照，确认数据准确性
5. **标注数据来源**：在任何使用抓取数据的场景中标注数据来源

### 应当遵守

1. **盘后运行**：排行接口在交易日 9:00-15:30 偶尔不返回，建议盘后运行
2. **合理批量**：批量抓取时控制总量，避免对平台造成过大压力
3. **及时更新**：数据有时效性，不应使用过期的抓取结果做决策
4. **关注告警**：校验产生的 warning 和 fatal 应认真对待，不应忽略

### 禁止行为

1. 禁止伪造数据（用大模型「常识」补全字段、生成示例数据充当真实结果）
2. 禁止绕过反爬机制（代理池、伪造 Cookie、逆向加密接口）
3. 禁止商业转售抓取数据
4. 禁止对数据源平台造成 DDoS 级别的访问压力
5. 禁止将本工具用于任何违法用途

## 法律风险提示

### 数据获取风险

- **违反平台使用条款**：高频爬取可能违反数据源平台的使用条款，平台有权封禁 IP 或采取法律行动
- **违反《数据安全法》**：大规模获取数据可能涉及数据安全问题，需遵守《中华人民共和国数据安全法》相关规定
- **违反《网络安全法》**：绕过反爬措施可能违反《中华人民共和国网络安全法》关于不得干扰网络正常功能的规定

### 数据使用风险

- **违反《著作权法》**：数据源平台对其数据编排享有著作权，商业转售可能侵犯著作权
- **违反《反不正当竞争法》**：利用抓取数据构建竞争性产品可能构成不正当竞争
- **违反《个人信息保护法》**：如数据涉及基金经理等个人信息，需遵守《个人信息保护法》相关规定

### 投资建议风险

- **违反《证券投资基金法》**：向公众提供基金投资建议可能需要相关资质，无资质提供可能违法
- **违反《证券法》**：提供证券投资咨询服务需要证监会批准的业务资格

### 免责边界

本免责声明不能免除使用者的法律责任。使用者应当：

1. 自行评估数据获取和使用的法律合规性
2. 在必要时咨询专业法律意见
3. 对自身行为承担全部法律责任
4. 如收到数据源平台的停止通知，应立即停止相关行为
